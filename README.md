## Requirements ##

* Python 2.6+
* uTorrent 3.0+

## How do I get set up? ##

* In uTorrent go to settings , advanced , run program , run program when torrent changes state
* input(with quotes):
* "Path\to\pythonw.exe Path\to\poisonprocess.py" %I %K %P %S
* edit included config.cfg file
