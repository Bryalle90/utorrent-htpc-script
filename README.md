# README #

## Program Requirements ##

### Required ###

* Python 2.7+
* uTorrent 3.0+

### Optional ###

* theRenamer

## How do I get set up? ##

* In uTorrent go to settings , advanced , run program , run program when torrent changes state
* input(with quotes):
                "Path\to\pythonw.exe" "Path\to\script.py" %I %K %P %S
* edit included config.cfg file
